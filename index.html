Google Sheets 설정 및 DB 연동 방법
  1. Google Sheets 새 파일 만들기
  2. 확장 프로그램 → Apps Script 클릭
  3. 기존 코드 전부 삭제 후 아래 코드 붙여넣기
  4. 배포 → 새 배포 → 웹앱 → 액세스: 모든 사용자
  5. 생성된 URL을 이 HTML 소스 최상단의 TEACHER_SHEETS_URL 변수 안에 붙여넣고 서버에 올리시면 완료됩니다.
